Программа, реализующая алгоритм Seam Carving для изменения размера изображения. Он находит такие seams(швы), которые содержат наименее важную информацию для изображения, и удаляет их.
Seams(швы) - бывают вертикальные и горизонтальные.
Вертикальный - кривая из пикселей, идущая сверху вниз, причем в каждом ряду по 1 пикселю, и каждый следующий пиксель является соседом пикселю сверху.
Горизонтальный - то же самое, только идет с право на лево.
Подробнее - http://en.wikipedia.org/wiki/Seam_carving

В данной программе реализовано:
-открытие изображений формата BMP 24 bit
-расчет энергии с помощью функции энергии двойного градиента
-удаление горизонтальных и вертикальных seam'ов c последующим сохранением в bmp файл
-все пока работает в консоле: сначала предлагается ввести название входного файла(по-умолчанию - input.bmp), затем указать сколько линий удалить.
После удаления будет предложено сохранить полученное изображение в файл (по-умолчанию - output.bmp);
